========================================================================
    CONSOLE APPLICATION : 08_service Project Overview
========================================================================

This project demonstrates the use of simple local servicing object to use for multithreading programming.
It creates dummy (empty) local service interface, which is not visible outside of the process.
The service is created as an instance of Component, Stub (dummy Stub) and Timer consumer.
The service is registration is done in mode, which is loaded at start and unloaded when application completes the job.
