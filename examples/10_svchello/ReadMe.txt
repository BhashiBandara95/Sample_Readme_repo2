========================================================================
    CONSOLE APPLICATION : 10_svchello Project Overview
========================================================================

This project demonstrates use of real local service, interface is described in interface design IDE and code is generated via tool.
The application creates local service component and the local client to communicate.
The local services are accessible only within single thread and are used for multi-threading communication.
The application initializes a local servicing component, so that it cannot be accessed outside of the process.
It demonstrates the use of request with and without response.
It as well demonstrates subscription features of AREG framework, where clients can subscribe / un-subscribe to certain messages during run-time.
