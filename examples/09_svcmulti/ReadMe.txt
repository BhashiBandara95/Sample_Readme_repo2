========================================================================
    CONSOLE APPLICATION : 09_svcmulti Project Overview
========================================================================

The project project multiplicates the duplication of the same service implementation by assigning different service names.
It is an extention of 08_service, uses empty services and insantiate 2 identical services with different names.
The same service implementation can be instantiated more than one time if every instantiated service has unique role name
and they are not running in the same thread. The multiplication is done in model.
